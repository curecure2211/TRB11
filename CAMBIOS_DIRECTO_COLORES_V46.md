# TRB v46

- Si Transbordos oficiales está activado y existe una ruta directa de un solo bus, se muestra la ruta directa antes de forzar un transbordo.
- Si no hay ruta directa, el filtro muestra únicamente transbordos oficiales permitidos.
- Corrige el color de empresas/rutas en el explorador de Rutas cuando el objeto de ruta no traía el nombre de la empresa.
- Normaliza los colores de cada ruta al cargar el catálogo para que el icono de empresa, las tarjetas de ruta y el trazado del mapa usen la misma paleta.
