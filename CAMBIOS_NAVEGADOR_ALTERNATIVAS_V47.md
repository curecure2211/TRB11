# TRB v47

- El contador de instrucciones ahora muestra solo las alternativas visibles del filtro activo.
- Si estás en Buses, el navegador muestra únicamente buses.
- Si estás en Transmetro, el navegador muestra únicamente Transmetro.
- Si están activos Transbordos oficiales, el navegador cuenta solo rutas directas permitidas o transbordos oficiales visibles.
- Las flechas anterior/siguiente ya no recorren las 30 alternativas mezcladas.
