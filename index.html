Privacidad

Diseñada sin rastreo

TRB no tiene un servidor propio de rastreo. Cuando buscas una dirección o calculas una caminata, el navegador consulta directamente servicios públicos basados en OpenStreetMap; la ubicación solo se solicita cuando pulsas “Mi ubicación”.

Los favoritos y la caché básica de direcciones se guardan en localStorage dentro del dispositivo.

Integración futura

Lista para datos oficiales

El planificador ya calcula opciones aproximadas con el catálogo disponible. Puede conectarse posteriormente a GTFS, GTFS-Realtime o una API oficial para añadir horarios, alertas, transbordos completos y vehículos en vivo.
