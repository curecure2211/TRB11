# Mapa vectorial de TRB

`trb-map-style.json` es una adaptación para Barranquilla del lenguaje visual del proyecto `hsl-map-style` aportado por el usuario.

- Renderizador: MapLibre GL JS.
- Esquema vectorial: OpenMapTiles.
- Proveedor por defecto: OpenFreeMap.
- Datos base: OpenStreetMap.
- Las rutas de transporte de Helsinki, zonas HSL y elementos de marca fueron eliminados.
- Los KMZ de TRB se convierten a GeoJSON y se dibujan como capas propias.

La adaptación conserva la licencia AGPL-3.0-only del estilo HSL. Consulte `HSL_STYLE_LICENSE_AGPL-3.0.md`.
