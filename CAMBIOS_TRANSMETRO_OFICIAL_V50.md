# TRB v50 — Transmetro + alimentadores y aliados oficiales

Cambios:

- Si Transmetro está activo, las combinaciones internas Transmetro + alimentador se mantienen aunque Transbordos oficiales esté activado.
- Activar Transbordos oficiales ya no reemplaza ni borra las conexiones propias de Transmetro.
- Transbordos oficiales agrega las conexiones Transmetro ↔ empresas aliadas autorizadas: Cooasatlán, Cootrab, Cootransnorte, Sobusa y Transoledad.
- Las conexiones Transmetro ↔ buses aliados oficiales aparecen aunque el switch Bus esté apagado, porque se tratan como parte del modo Transmetro oficial.
- Las rutas directas siguen apareciendo primero, pero ya no ocultan las demás alternativas válidas.
