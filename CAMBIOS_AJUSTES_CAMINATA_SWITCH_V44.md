# TRB v44

- Las sugerencias de buses y combinaciones priorizan menor caminata inicial, menor caminata de transbordo y menor caminata final.
- Se mantiene el límite máximo de 2 transbordos.
- El activador de Transbordos oficiales ahora es un switch ON/OFF estilo Apple, azul cuando está activo.
- Si Transbordos oficiales está activo, el filtro se mueve a Todas para mostrar únicamente las combinaciones oficiales disponibles.
- El panel Ajustes se abre junto al panel de búsquedas, no en el extremo derecho del mapa.
