# TRB v22 — rutas cercanas y combinaciones inteligentes

- La geocodificación prioriza lugares específicos y evita confundir un destino con el departamento del Atlántico.
- Se invalida la caché antigua de direcciones para corregir resultados previamente guardados.
- Los KMZ se muestrean con una densidad aproximada de 140 m para detectar rutas realmente cercanas.
- Primera pasada con accesos cortos; segunda pasada con cobertura ampliada cuando el destino está fuera de la red inmediata.
- Hasta dos transbordos entre Transmetro y las 23 empresas.
- Radio final ampliado para destinos costeros o conectores como Las Flores–Puerto Mocho.
- Las alternativas caminando y en bicicleta se conservan, pero ya no sustituyen rutas de transporte viables.
