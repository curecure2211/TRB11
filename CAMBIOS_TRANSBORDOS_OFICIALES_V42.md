# TRB v42

- Agrega en Ajustes el botón **Transbordos oficiales**.
- Integra las reglas oficiales de transbordo del resumen SIBUS/Transmetro.
- Cuando está activo, el planificador muestra solo combinaciones oficiales con tarifa reducida.
- Cada tarjeta oficial muestra segundo pasaje, total lunes-sábado, total domingos/festivos y tiempo máximo de conexión.
- Se generan explícitamente planes para las parejas oficiales de rutas para evitar depender solo de combinaciones generales.
